# IELTS Writing Task 1 - High Scoring Essays (V1 Raw Extracted)

This directory contains the raw extracted IELTS Writing Task 1 essays with high scores (band score 9).

## Data Source

The data in this directory was extracted from the Hugging Face dataset:
[hai2131/IELTS-essays-task-1](https://huggingface.co/datasets/hai2131/IELTS-essays-task-1)

## Contents

This version contains raw extracted data where each entry represents a high-scoring (band score 9) IELTS Writing Task 1 essay along with its evaluation metrics.

## Processing

These files were extracted using the [extract_high_scores.py](../extract_high_scores.py) script from the main repository, which filters entries with an overall band score of 9 from the original dataset.

## Structure

Each entry contains:
- Topic category
- Subject matter
- Image URL (for chart/graph questions)
- Image description
- Essay content
- Evaluation scores (Overall, Task Response, Coherence & Cohesion, Lexical Resource, Grammatical Range & Accuracy)
- Detailed descriptions for each score category
