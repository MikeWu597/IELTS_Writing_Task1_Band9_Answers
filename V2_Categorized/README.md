# IELTS Writing Task 1 - High Scoring Essays (V2 Categorized)

This directory contains the categorized IELTS Writing Task 1 essays with high scores (band score 9), organized by topic type.

## Data Source

The data in this directory was extracted from the Hugging Face dataset:
[hai2131/IELTS-essays-task-1](https://huggingface.co/datasets/hai2131/IELTS-essays-task-1)

## Contents

This version contains the same high-scoring (band score 9) IELTS Writing Task 1 essays as V1, but organized into topic-specific folders for easier navigation and study.

## Processing

These files were processed using the [categorize_pdfs.py](../categorize_pdfs.py) script from the main repository, which groups the essays by their topic type and moves them into corresponding directories.

## Categories

Essays are organized into folders based on their topic types, which may include:
- Line Graph
- Bar Chart
- Pie Chart
- Table
- Map
- Process Diagram
- Mixed Charts
- Other topic types present in the dataset

Each folder contains PDF files with the essays belonging to that specific topic type.
